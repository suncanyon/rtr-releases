# rtr-releases

Public binary releases for [RTR CLI](https://github.com/suncanyon/rtr).

## Install

```sh
# Shell installer (macOS & Linux)
curl -fsSL https://install-rtr.ext.suncanyontech.com/install.sh | sh

# Homebrew
brew install suncanyon/rtr/rtr
```

Releases are published automatically when a new version is tagged in the source repo.
